# Система управления заказами в интернет-магазине

Приложение разработано на языке Kotlin с фреймворком Spring. 
Приложение предоставляется GraphQL API для управления заказами.
GraphQL схема расположена в 
[lab3/src/main/resources/graphql/schema.graphqls](src/main/resources/graphql/schema.graphqls).
Примеры GraphQL запросов в [lab3/queries](queries).

GraphiQL конструктор доступен по [localhost:8080/graphiql](http://localhost:8080/graphiql).
GraphQL эндпоинт - стандартный. 

1) Создайте связанные между собой data-классы:
   - Продукт: название, категория, цена, количество на складе.
   - Клиент: имя, контактные данные, история заказов.
   - Заказ: список продуктов, клиент, статус заказа.
2) Используя коллекции и лямбда-выражения, создайте функции для:
   - Добавления продуктов в каталог, обновления информации о наличии.
   - Регистрации клиентов.
   - Оформления заказов клиентами с обновлением остатков на складе.
   - Расчёта общей стоимости заказа с учётом возможных скидок.
   - Поиска продуктов по категории, цене.
   - Управления корзиной покупателя.
   - Анализа продаж по категориям продуктов.
3) Создайте приложение, эмулирующее процесс покупки в интернет-магазине, включая
   просмотр каталога, добавление товаров в корзину и оформление заказа.
